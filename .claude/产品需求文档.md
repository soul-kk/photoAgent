# AI摄影助手需求文档

## 一、用户需求与痛点分析

### 1.1 核心痛点

摄影初学者在学习摄影过程中普遍面临以下问题：

- **"拍不出感觉"**：虽然掌握了基本的拍摄技巧，但无法拍出具有美感和艺术性的照片
- **"不知道如何改进"**：拍摄后无法准确判断照片的问题所在，缺乏专业的指导建议
- **"后期调色迷茫"**：不懂得如何通过后期处理提升照片质感，无法实现想要的影调风格
- **"选片困难"**：拍摄多张照片后，难以客观判断哪张效果最好
- **"拍前没有方向"**：到达拍摄场景后不知道如何构思机位和焦段，缺乏专业的前期规划

### 1.2 用户期望

- 获得专业级的摄影分析和改进建议
- 拍摄前获得针对性的机位和构图建议
- 快速提升摄影审美和技术水平
- 高效完成选片工作

## 二、产品功能

### 2.1 AI摄影分析【核心功能】

**功能描述**：用户上传单张图片，系统自动进行全方位分析

**核心能力**：

- 构图分析：识别构图方式（三分法、对称构图、引导线等），评估构图合理性
- 色彩分析：分析色彩搭配、色调倾向、色彩饱和度等
- 内容识别：识别拍摄主体、场景类型、画面元素
- 曝光评估：判断曝光是否准确，识别过曝或欠曝区域

**输出内容**：

- 详细的图文分析报告
- 具体的改进建议
- 自动生成的"改进示意图"（标注问题点和优化方向）

### 2.2 AI拍摄建议【核心功能】

**功能描述**：用户在正式摄影前，上传一张当前所处场景的照片，并告知 AI 想要拍摄的主体，AI 根据场景特点给出专业的前期拍摄建议

**核心能力**：

- 场景解读：分析场景光线、空间结构、背景元素、环境氛围
- 主体理解：结合用户描述的拍摄主体，判断最佳表现方式
- 机位建议：推荐拍摄角度（正面/侧面/俯拍/仰拍等）及站位距离
- 焦段建议：根据场景和主体推荐适合的焦段范围（广角/标准/中长焦）
- 拍摄要点：提示构图技巧、对焦点选择、景深控制、光线利用等关键注意事项

**输出内容**：

- 推荐机位示意（文字描述 + 场景标注图）
- 建议焦段及理由
- 3～5 条具体拍摄要点
- 可选的备选方案（不同风格/创意的拍法）

### 2.3 AI图片质量判断【拓展功能，暂不实现】

**功能描述**：用户上传多张图片，系统智能对比并推荐最佳照片

**核心能力**：

- 多维度对比：构图、色彩、曝光、清晰度、创意性等
- 智能排序：根据综合评分对图片进行排序
- 最佳推荐：自动选出质量最高的一张图片

**输出内容**：

- 最佳照片推荐
- 详细的选择理由说明
- 其他图片的优缺点分析

### 2.4 影调风格建议【拓展功能，暂不实现】

**功能描述**：用户通过文字描述想要的影调风格，系统生成后期参数建议

**核心能力**：

- 自然语言理解：理解用户对风格的描述（如"王家卫电影感""富士清冷色系"）
- 参数生成：生成对应的后期调整参数

**输出内容**：

- 风格匹配结果
- 具体的后期参数建议（曝光、对比度、饱和度、色调、色温等）
- 参数调整的前后对比示意图

## 三、核心技术

### 3.1 图像识别与理解

- **目标检测**：识别图片中的主体物体和关键元素
- **场景分类**：判断拍摄场景类型（风景、人像、街拍等）
- **构图分析**：识别构图方式和构图质量评估
- **色彩分析**：提取色彩特征，分析色彩搭配和色调倾向

### 3.2 提示词工程

- **精准提示词设计**：针对不同摄影场景设计专业的提示词模板
- **多轮对话优化**：通过迭代优化提示词，提升AI分析准确性
- **风格化描述**：将影调风格转化为AI可理解的提示词

### 3.3 大语言模型应用

- **自然语言理解**：理解用户对拍摄主体和影调风格的描述
- **专业建议生成**：生成专业的摄影改进建议、拍摄建议和选片理由
- **场景推理**：根据场景图像推断最优拍摄策略

### 3.4 图像生成与编辑

- **改进示意图生成**：基于分析结果生成标注示意图
- **机位标注图生成**：在场景图上标注推荐机位和拍摄方向
- **风格化渲染**：生成不同影调风格的预览效果
- **参数映射**：将风格描述转化为具体的后期参数
